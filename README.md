# 🌍 Global Citizens Community Center (GCCC)

A multi-page website for the **Global Citizens Community Center** — a nonprofit community hub aligned with all 17 UN Sustainable Development Goals, operating across 14 themed rooms and serving a growing global membership.

---

## 📁 File Structure

```
index.html           → Landing page (home)
gccc-pages.html      → Member dashboard + 14 room detail pages (SPA)
fundraiser.html      → Fundraiser launch page
community-center.html → Community center portal (GlobalHive)
back-office.html     → Staff & volunteer back office (password-protected)
README.md            → This file
```

## 🚀 Deployment (GitHub Pages)

1. Push all files to a GitHub repository (e.g. `gccc-website`)
2. Go to **Settings → Pages**
3. Set **Source** to `main` branch, `/ (root)` folder
4. Your site will be live at `https://<your-username>.github.io/gccc-website/`

> **Important:** All pages use relative links (e.g. `href="gccc-pages.html"`), so the site works correctly at any domain root.

## 🧭 Features

- **Landing Page (`index.html`):** Hero, Impact stats, Live Network (YouTube + Jitsi), Programs, 14 Room Cards, SDG Goals grid, World Data, Campaigns, Sustainability Hour, Global Summit, Membership tiers, SDG Game
- **Member Dashboard (`gccc-pages.html`):** Single-page app with 6 tabs (Overview, Activity, Donation History, Event Calendar, GlowBag Sponsorship, Video Call) and dynamic room detail pages for all 14 rooms
- **Fundraiser (`fundraiser.html`):** No Kid Hungry–linked campaign launcher, 6 fundraiser types, 4-step guide, campaign submission form
- **Community Center (`community-center.html`):** GlobalHive portal with live streaming, rooms browser, and community features
- **Back Office (`back-office.html`):** Staff dashboard with schedule, directory, training tracker, and room management
